This project focuses on performing Exploratory Data Analysis (EDA) on a student dataset using Python. The goal is to understand data patterns, identify key insights, and prepare the dataset for future machine learning or analytics tasks.

 Project Overview

The notebook includes:

Data loading and inspection

Handling missing values

Data cleaning and formatting

Descriptive statistics and summary exploration

Visualization of key patterns and relationships

Identification of outliers

Data-driven insights and conclusions
